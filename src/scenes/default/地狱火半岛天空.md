# 给默认场景增加地狱火半岛天空
**参考demo.png**
## 需求描述：
这是一个典型的 魔幻风格的天空极光 / 空气流动能量带（arcane aurora）。

它有这些特征：

✅ 1. 条带状的大规模光流

带有方向性，像风一样被拉伸。

✅ 2. 颜色渐变：绿色 → 黄色 → 淡橙色

保持柔和过渡。

✅ 3. 半透明叠加效果

你能看到背景的天空颜色透出。

✅ 4. 动态模糊 + 扭曲的噪声纹理

它不像真实云，而是典型游戏风格的“能量带”。

🎮 类似于：

游戏里常见的 skybox + aurora shader（天空盒 + 极光着色器）组合。

我告诉你一个 成熟稳定的制作方案，跟游戏引擎做法一模一样，而且效果绝佳。

它由三个层（Layer）组成：
第 1 层：整张天空背景（Skybox）

最底层是一个简单的天空颜色渐变。

Babylon.js 推荐用 SkyMaterial，它负责图中那种紫色渐变背景。

第 2 层：极光（核心特效）

这里是重点！🔥

要实现类似“地狱火半岛黄色能量流”的效果

你需要使用 ShaderMaterial + 两层噪声纹理 + 动态 UV 扭曲。
原理（超级关键）

极光效果本质是：

（基础渐变）×（噪声遮罩）×（方向性拉伸）×（半透明叠加）

Babylon.js 非常适合这个结构！
你可以用一个大平面（巨型 quad）贴在天空上方

这样你不需要破坏 skybox。
用 ShaderMaterial 做动态极光
片段着色器（Aurora 的核心逻辑）

核心是 分形噪声 fbm（fractial brownian motion）。
这样你就能得到：

条带状

半透明

随风流动

柔光型

魔法能量感

跟图里的一模一样！
加入扭曲运动（风吹效果）

第 3 层：模糊光效（后处理 Bloom）

为了让黄色更亮、更“能量感爆表”，你应该打开 Bloom。
效果极其明显，会产生图中这种：

光晕扩散 → 会把能量带渲染得非常梦幻

🎉 组合在一起的效果

按照上面的 3 层：

SkyMaterial：紫色天空背景

Aurora Shader：黄色/绿条带状能量流

Bloom：光晕效果强化能量感

最终效果就会非常接近你截图里的那种地狱火半岛天空的魔幻风格！
