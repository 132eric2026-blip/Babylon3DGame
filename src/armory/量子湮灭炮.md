# 量子湮灭炮 (Quantum Annihilator)

## 武器设计理念
量子湮灭炮是一款高科技能量武器，利用量子湮灭原理释放毁灭性的能量束。其设计融合了未来科技美学与量子物理概念。

## 视觉特征

### 配色方案
- **主色调**：深蓝色金属（科技感）
- **能量色**：蓝色、紫色、青色渐变
- **发光效果**：强烈的蓝紫色能量光晕

### 核心组件
1. **六边形主炮身** - 深蓝色金属材质，科技感十足
2. **量子核心** - 中央蓝色能量球，带脉动动画
3. **三层旋转能量环** - 蓝紫交替，多轴向旋转
4. **多段式炮管** - 前端聚焦设计
5. **能量导管** - 4条青色螺旋管道
6. **量子稳定器** - 后部八面体装置，持续旋转
7. **电弧装饰** - 6个闪烁的青色能量球

## 特效系统

### 地面光环
- **量子符文阵列**：六芒星 + 三角形 + 同心圆
- **颜色**：蓝紫渐变光晕
- **动画**：持续旋转

### 粒子效果
1. **螺旋量子粒子**
   - 蓝色/紫色/青色混合
   - 螺旋上升运动
   - 发射率：80粒子/秒

2. **电弧闪电粒子**
   - 青色电弧效果
   - 球形发射
   - 快速闪烁

### 光照效果
- **点光源**：蓝紫色动态光照
- **强度**：1.5-2.5 闪烁变化
- **范围**：10单位

## 动画效果
- 量子核心脉动（缩放1.0-1.15）
- 三层能量环多轴旋转
- 稳定器全方位旋转
- 电弧节点闪烁
- 武器整体浮动 + 旋转
- 光环地面旋转

## 技术实现
- 使用 Babylon.js 引擎
- 双粒子系统（主粒子 + 电弧粒子）
- 多重动画系统
- 动态纹理生成（量子符文）
- 点光源动态效果
