# 半月斩技能 - 快速上手指南 🌙⚔️

## 🎮 如何使用

### 第一步：启动游戏
```bash
npm run dev
```

### 第二步：进入游戏
打开浏览器，访问本地开发服务器（通常是 `http://localhost:5173`）

### 第三步：找到技能栏
- 查看屏幕底部的技能栏
- 找到第3个技能格（显示数字"1"）
- 看到青白色的半月形图标 ✅

### 第四步：释放技能
- 按下键盘数字键 **`1`**
- 观察玩家前方2米处出现青白色半月形气波
- 气波快速从小变大，在原地停留
- 伴随大量粒子特效和火花飞溅

### 第五步：再次施放
- 等待 3 秒冷却时间
- 再次按下 `1` 键即可重复释放

## 🔍 效果详解

### 视觉特效
- **半月形气波**：明亮的青白色，双面发光，在玩家前方2米处停留
- **粒子效果**：400粒子/秒的光辉 + 250粒子/秒的火花
- **动画效果**：快速放大、停留、缩小淡出，轻微旋转
- **发光层**：强烈的辉光效果

### 技能参数
- **冷却时间**：3秒
- **释放距离**：玩家前方2米
- **持续时间**：0.8秒
- **半径**：2.5米
- **最大缩放**：1.3倍

## 🎯 技巧与建议

1. **朝向很重要**：气波会沿着角色面向的方向释放，调整好方向再释放
2. **移动施法**：可以在移动、跳跃时释放技能
3. **近距离攻击**：气波在玩家前方2米处出现，适合近距离战斗
4. **視觉享受**：在暗处释放效果更佳（发光效果更明显）

## 🐛 故障排除

### 问题：按下数字键1没有反应
- 检查控制台是否有错误信息
- 确认游戏已完全加载
- 尝试点击游戏画面获取焦点

### 问题：看不到气波效果
- 检查浏览器是否支持 WebGL 2.0
- 尝试调整相机角度观察
- 查看控制台是否有 "激活技能：半月斩" 的输出

### 问题：技能一直提示冷却中
- 等待完整的3秒冷却时间
- 检查控制台的冷却剩余时间提示

## 📊 性能说明

- **CPU占用**：低（单次技能仅创建一个网格）
- **内存占用**：低（0.8秒后自动清理）
- **GPU占用**：中等（粒子系统和发光效果）
- **网络占用**：无
