Babylon.js 影斩（Shadow Slash / Dark Slash）
—— AI 代码生成提示词（Markdown）
🎯 目标

生成一个基于 Babylon.js 的“影斩 / 暗影斩”技能特效，包括：

黑紫色暗能量刀光

类似空间撕裂的裂缝效果

暗影粒子（黑色烟雾、紫光碎片）

残影 + 空间扭曲视觉

可调用技能函数

可独立运行的完整 Demo

🧩 核心功能要求

请生成完整 Babylon.js 代码（JS/TS 均可），实现以下能力：

1. 场景 + 角色

自动创建 engine、scene、camera、light

创建地面

加载 GLB 角色模型（若无则自动创建人形占位）

附带简单攻击动作（可以是占位动画）

2. 影斩技能主函数

定义：

castShadowSlash(character, direction)


动作流程：

角色进行快速挥砍

在角色前方出现一条 暗影刀光（Shadow Blade）

刀光呈现：

黑紫色

半透明

带能量边缘

中心暗色、外圈发光

刀光根据方向高速挥出

随后刀光在空中“撕裂空间”般裂开 → 再迅速闭合消失

🌑 3. 暗影刀光视觉规格
材质与色彩

主色：深紫（RGB 0.3, 0.05, 0.4）

边缘发光（GlowLayer）

中心部分透明度更高

混合：ALPHA_ADD 或 ALPHA_COMBINE

刀光形状

可使用：

自定义半月形 Mesh

Plane + Shader

Ribbon + 动态扰动

Shader 要求（若使用）：

UV 扭曲（体现空间扭裂）

边缘噪声（提升魔法感）

中间黑洞风格 radial fade

🕳️ 4. 暗影裂缝（空间撕裂特效）

在刀光路径中生成：

一个或多个黑色裂缝贴图（用一张噪声贴图）

轻微深度偏移模拟“空间破碎”

裂缝边缘发光（紫色）

裂缝在 0.3 秒内自动闭合消失

💀 5. 暗影粒子与拖尾

粒子组成：

黑色雾状粒子（低透明度）

紫色能量碎片

少量亮点像星屑（white / violet）

粒子效果：

从刀光边缘喷出

随着刀光移动被吸入和散射

生命周期短，体现“诡异破裂”

拖尾：

使用 TrailMesh 或 Ribbon

颜色从紫 → 黑渐变

尾部透明度逐渐降低

长度与方向随角色攻击方向变化

按 K 键 释放影斩

技能可连发（无冷却）
